    SSL TLS sertificates

    creting certificates

    private key openssl genrsa -out ca.key 2048 specify name of what is for. this is signing openssl req -new -key ca.key -subj "/CN=KUBERNETES-CA" -out ca.csr sign request. this is self signed openssl x509 -req -in ca.csr -signkey ca.key -out ca.crt

    generate client certificeates

    openssl genrsa -out admin.key 2048 this name is for logs mostly openssl req -new -key ca.key -subj "/CN=kube-admin" -out admin.csr we must mention group details in signing request openssl x509 -req -in admin.csr -CA ca.crt -CAkey ca.key -out admin.crt

    why we need certs

    we cen auth to kluster apiserver using this key or in cluster definiton we can add this keys
